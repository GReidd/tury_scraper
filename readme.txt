Для того, чтобы заработал парсер нужно:
1. Установить Python 3.x (проверить установлен ли Python можно командой python --version в командой строке на компьютере, команда должна выдать версию)
2. Установить нужные библиотеки с помощью файла requirements.txt командой pip install -r requirements.txt.
